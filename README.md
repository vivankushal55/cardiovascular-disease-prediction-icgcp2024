## Abstract

Cardiovascular disease remains the leading cause of global mortality, accounting for nearly one-third of all deaths. This research leverages advanced machine learning and data mining techniques to predict the occurrence of heart diseases, supporting healthcare professionals in rapid diagnosis.

## Novel Contribution: QMBC

We introduce the **Quine-McCluskey Binary Classifier (QMBC)** — a novel ensemble technique that integrates seven distinct algorithms:

- Logistic Regression
- Decision Tree
- Random Forest
- K-Nearest Neighbor
- Naive Bayes
- Support Vector Machine
- Multilayer Perceptron

## Performance

The QMBC technique significantly enhances heart disease prediction accuracy across:

- Heart Valve Disease
- Coronary Artery Disease
- Heart Arrhythmias
- Pericardial Disease
